Endpoints Fetch Web Data

Fetch Web Data

GET /v2/fetch

The fetch endpoint allows you to retrieve the full rendered HTML content of any webpage. It handles JavaScript execution, CSS rendering, and complex cookies automatically. Use this for single-page applications (SPAs) or sites requiring heavy browser interaction.

Request Parameters

url string Required

The absolute URL of the page you want to scrape. Must include protocol (http/https).

render_js boolean Default: true

Wait for the page to finish executing JavaScript before returning the content.

proxy_country string

ISO 3166-1 alpha-2 country code for the egress IP address (e.g., 'US', 'GB', 'DE').

Response Object

Property Type Description
content string The raw HTML content of the page.
status integer HTTP status code returned by target.
metadata object Includes response time and proxy info. 
import requests

# API Configuration
url = "https://api.scrapyx.io/v2/fetch"
headers = {
    "X-API-KEY": "YOUR_API_KEY"
}
params = {
    "url": "https://example.com",
    "render_js": True,
    "proxy_country": "US"
}

# Execute Request
response = requests.get(url, headers=headers, params=params)

# Output results
print(response.json()['content'])
